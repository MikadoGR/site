Memocord

Welcome to the description page of Memocord. Memocord allows users to create posts and share them via ID with each other. You can also view all posts using specific commands. Below you'll find a list of commands and their descriptions.

  • /help

    Displays a list of available commands and their usage.

  • /info

    Provides information about Memocord.

  • /ownerinfo

    See how many servers the bot is currently in.

  • /ping

    Checks the bot's latency.

  • /post create

    Creates a new post.

  • /post comment

    Adds a comment to a post.

  • /post delete

    Deletes a specified post.

  • /post homepage

    Shows the homepage with all posts.

  • /post view

    Views a post using its ID.

  • /post rules

    Displays the rules for posting.

  • /post color

    Shows you a site to grab a color hex code.
