Memocord

Welcome to Memocord. Memocord allows users to create posts and share them via ID with each other. Below you'll find a list of commands and their descriptions.

  • /help

    Displays a list of available commands and their usage.

  • /info

    Provides information about Memocord.

  • /ownerinfo

    Gives information about the bot's owner.

  • /ping

    Checks the bot's latency.

  • /post create

    Creates a new post.

  • /post comment

    Adds a comment to a post.

  • /post delete

    Deletes a specified post.

  • /post homepage

    Shows the homepage with all posts.

  • /post view

    Views a post using its ID.

  • /post rules

    Displays the rules for posting.
